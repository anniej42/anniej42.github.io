Collaborative Whiteboard

December 9, 2013
  • Java
  • Sever
  • GUI

Screenshot of the GUI in action

As the final project in 6.005, my teammates and I worked on a collaborative whiteboard project. A server hosts multiple whiteboards, and a whiteboard supports multiple users. Each client can create a new board or join an existing one, and they can work simultaneously on the same whiteboard.

Some cool features that our whiteboard supports are: switching to a new board from existing board, customized color picker, drag from pallet to change background color, post to Facebook, etc.
